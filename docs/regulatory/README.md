# Regulatory Documentation

This folder contains FCA Handbook excerpts and internal requirements for the Geopolitical Health Scan feature.

## Purpose

The Geopolitical Health Scan is an **internal-only** decision-support tool for financial advisers and compliance teams. It is NOT client-facing and does NOT provide financial advice or recommendations.

## Contents

- **FCA_EXCERPTS.md**: Extracted text from relevant FCA Handbook sections
- **REQUIREMENTS.md**: Internal "Do/Don't" cheat sheet derived from FCA requirements

## Key Regulations

- **COBS 9**: Suitability requirements (including basic advice)
- **SYSC 7**: Risk control systems and controls
- **Consumer Duty**: Good outcomes and foreseeable harm
- **DISP**: Complaints handling and evidence requirements

## Usage

These documents are used by:
1. The regulatory retriever to provide context to the LLM prompt
2. Compliance teams to validate output language
3. Developers to ensure non-directive, audit-ready outputs
